# Wing-Spar-Static-Analysis

## Objective

Design and perform finite element analysis of an aircraft wing spar under aerodynamic loading conditions.

## Tools Used

- Fusion 360 — CAD modelling
- Abaqus — Finite Element Analysis
- AutoCAD — Technical drawings

## Methodology

1. Created wing spar geometry
2. Defined material properties
3. Applied loading and boundary conditions
4. Generated finite element mesh
5. Evaluated structural performance

## Results

Analysis includes:

- Von Mises stress distribution
- Total deformation
- Factor of safety evaluation

## Conclusion

The wing spar design was evaluated for structural integrity and weight efficiency.
